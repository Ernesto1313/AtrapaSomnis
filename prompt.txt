## Contexto
App Android "AtrapaSomnis" (package: com.ernesto.atrapasomnins).
Diario personal de sueños. Sin internet, sin APIs externas, sin Room.
Los datos se guardan en fichero JSON en almacenamiento interno.
Kotlin + Jetpack Compose, MVVM, Navigation Compose.
Comentarios en español, naturales, como los escribiría un estudiante.

### 4. Crear AtrapaSomnisApplication.kt

Ruta: app/src/main/java/com/ernesto/atrapasomnins/AtrapaSomnisApplication.kt

package com.ernesto.atrapasomnins

import android.app.Application
import dagger.hilt.android.HiltAndroidApp

// Clase principal de la aplicación, necesaria para que Hilt funcione
@HiltAndroidApp
class AtrapaSomnisApplication : Application()

### 5. Actualizar AndroidManifest.xml

Añadir dentro de <manifest>:
    android:name=".AtrapaSomnisApplication"
    en la etiqueta <application>

Añadir permisos necesarios antes de <application>:
    <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
    <uses-permission android:name="android.permission.ACCESS_COARSE_LOCATION" />
    <uses-permission android:name="android.permission.POST_NOTIFICATIONS" />
    <uses-permission android:name="android.permission.SCHEDULE_EXACT_ALARM" />

Asegurarse de que el tema de la app apunta a:
    android:theme="@style/Theme.AtrapaSomnis.Splash"
